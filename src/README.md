## 前言

本配置是根据前人总结结合个人使用后的一套修改品。

## 用法

#### npm安装

```bash
npm i iapyang-eslint babel-eslint eslint -D
```

#### 新建`.eslintrc`文件

```javascript
{
    "extends": "./node_modules/iapyang-eslint/.eslintrc",
    "rules": {
      	// 此处添加或复写你自己的rules，以下是事例
        "no-underscore-dangle": 0,
        "prefer-rest-params": 0,
        "no-plusplus": 0,
        "max-len": 0
    }
}
```
